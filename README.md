**HealthStack System** is a full-stack healthcare management platform built using **Django REST Framework** and **React.js**. It digitalizes hospital operations and provides secure dashboards for **patients, doctors, and administrators**. The system handles online appointment booking, electronic medical records (EMR), prescriptions, billing, and real-time status updates.
